# Задание 1. Карта возможностей и матрица решений Buy vs Build

## Текущее состояние

VitaStore несёт избыточные расходы из-за уникальных процессов в каждом магазине.
Локальные POS, базы данных, расчёт премий, справочники и драйверы оборудования являются проблемными зонами:
они дороги в сопровождении, но не создают конкурентного преимущества.

Конкурентным преимуществом сети является Smart Fulfillment и связанная с ним
омниканальная логистика. Эти преимущества, а также специфичные правила
ассортимента, следует развивать внутри компании.

## Принципы перехода

1. Централизовать управление ассортиментом, ценами, промо и лояльностью.
2. Удалить постоянные локальные серверы и платные серверные лицензии.
3. Изолировать Cloud POS от моделей оборудования через Edge HAL.
4. Автоматизировать M&A-онбординг, оставив человеку только проверку исключений.
5. Не переписывать SAP ERP: интегрировать его через контролируемый API-слой.
