# Задание 4. TCO Analysis

## Итог

На горизонте трёх лет для 100 магазинов:

- TCO As-Is: **200,8 млн руб.**
- TCO To-Be: **107,8 млн руб.**
- экономия: **93,0 млн руб.**, или **46,3%**
- ROI миграции: **620%**

Переход экономически выгоден и достигает цели снижения TCO минимум на 40%.
Даже при росте стоимости облачных сервисов или миграции на 20% экономия остаётся
выше 44%.

## Условие достижения эффекта

Экономия возникает только после фактического вывода локальных серверов,
серверных лицензий и соответствующей нагрузки на команду поддержки. Если оставить старую
инфраструктуру в качестве постоянного запасного варианта, целевой эффект достигнут не будет.
