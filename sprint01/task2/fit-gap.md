# Fit/Gap анализ

## Список систем

В данном разделе описаны пробелы между текущими возможностями систем
и требованиями для проведения цифрового аукциона. Для первого аукциона
используется промежуточное решение без существенных доработок существующих ИС:
недостающие данные готовятся в Excel по утвержденным шаблонам и публикуются
в частном облаке.

| Система | Функциональность | Комментарий |
| --- | --- | --- |
| ИС Алмазы (существующая) | Отбор алмазов для онлайн-продаж (новая) | Необходимо внедрить функциональность маркировки лотов, предназначенных для цифрового канала, формирование цифровых ведомостей |
| ИС Логистика (существующая) | Управление маршрутами Офис-Лаборатория-Офис (новая) | Необходимо добавить новые этапы жизненного цикла груза, статусы перемещения камней между офисом и внешней лабораторией |
| ИС Аукцион (существующая) | Публикация профилей, проведение онлайн-торгов (новая) | Необходимо разработать интерфейс для отображения результатов оцифровки, обеспечить стабильную работу онлайн-аукциона |
| ИС Лаборатория (новая) | Оцифровка камней (новая) | Необходимо создать систему для хранения и первичной обработки данных о характеристиках камней |
| ИС Бухгалтерия (существующая) | Учёт новых расходов (новая) | Необходимо настроить новые статьи расходов (новые услуги лаборатории), автоматизировать сверку актов с внешними подрядчиками |
| Облачные сервисы | Публикация и хранение данных для клиентов (новая) | Необходимо организовать безопасную среду для размещения визуализации камней, описания их характеристик |

## Список интеграций

В данном разделе описаны пробелы в интеграциях между системами и то, как они будут реализованы.

| Системы | Способ передачи | Описание данных |
| --- | --- | --- |
| Алмазы ←→ Логистика (существующая) | Вручную (бумажные накладные, Excel-файлы) | Планы отгрузок, списки камней (существующие данные) |
| Лаборатория → Аукцион (новая) | Вручную (загрузка файлов в облако или систему) | Визуализация и оцифрованные характеристики (новые данные) |
| Лаборатория → Бухгалтерия (новая) | Вручную (сканы актов, ручной ввод сумм) | Стоимость услуг за каждый камень (новые данные) |
| Логистика → Бухгалтерия (существующая) | Вручную (загрузка маршрутных листов) | Стоимость транспортных услуг (существующие данные) |
